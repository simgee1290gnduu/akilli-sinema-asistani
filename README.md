# 🎬 Bulanık Mantık ile Akıllı Sinema İçerik Öneri Asistanı

Bu proje, **Mamdani Bulanık Çıkarım Motoru** kullanarak kullanıcı tercihlerine ve sinematik bağlamlara göre dinamik film/içerik öneri skorları üreten akıllı bir karar destek sistemidir.

## 🏗️ Sistem Mimarisi
Sistem, toplam **13 giriş parametresini** (IMDb, Trend, Tür Eşleşmesi, Dil Uyumu vb.) ve **27 harmanlanmış kuralı** işleyerek anlık reaktif sonuçlar üretir.
